Plugin to integrate the Course Booking System (Course Finder) into Wordpress

== Description ==

Course Finder is an online booking system for courses, seminar, events and appointments.
This plugin allows you to integrate the Course Booking System (Course Finder) into your wordpress site. Hence, you can setup one time or recurring courses that your participants can easily book through the web form that is automatically generated and embedded into you page or post using the short code. Your customers can then book and pay for the courses at any time.

It takes only five minutes to setup, and your participants/clients an immediately start booking your courses:
1) Sign-up/create an account or login into Course Finder.
2) Create a course or appointment type with description etc. 
3) Add sessions to the course, i.e., when and where the course will take place. For appointments, simply define your availability.
4) Use the generated short code to embed the auto generated form in any page or post of your wordpress site.

Your customers can now book the courses conveniently through your webpage.

== Installation ==

1. Log in to your WordPress admin panel and go to Plugins -> Add New
2. Type **Course Booking Platform** in the search box and click on search button.
3. Find Course Booking Platform.
4. Then click on Install Now after that activate the plugin.

OR

1. Download and save the **Course Booking Platform** plugin to your hard disk.
2. Login to your WordPress and go to the Add Plugins page.
3. Click Upload Plugin button to upload the zip.
4. Click Install Now to install and activate the plugin.
